# 🏠 Moscow Flats Pipeline

Пайплайн для сбора, хранения и аналитики рынка недвижимости Москвы.
Обеспечивает полный цикл обработки данных: от парсинга объявлений до построения витрин и обучения ML-модели для оценки стоимости квадратного метра.

## 🛠️ Основной стек
* **Оркестрация:** AirFlow
* **Хранилище:** PostgreSQL (DWH), S3 (MinIO)
* **Compute:** DuckDB
* **Контейнеризация:** Docker, Docker Compose
* **BI:** MetaBase
* **ML:** CatBoost


## 🐍 Используемые библиотеки Python
* `logging`
* `datetime`, `pendulum`
* `json`
* `duckdb`
* `pandas`, `numpy`
* `playwright`, `bs4`
* `boto3`


## 🏗️ Архитектура данных
Используется Medallion Architecture

* Raw Layer (S3): Сырые данные в формате **JSONL**, поступающие напрямую от парсера без изменений

* Silver Layer (S3): Очищенные, типизированные и дедублицированные с помощью `DuckDB` данные в формате **parquet**

* Gold Layer (PG): DWH с историческими данными по `SCD2` и готовые витрины для `MetaBase`.


## 🚀 Быстрый запуск
1. **Клонирование репозитория:**
    ```
    git clone https://github.com/Ilyasir/moscow-flats-pipeline.git
    cd moscow-flats-pipeline
    ```
2. **Настройка окружения:**
Скопируйте пример файла .env.
    ```
    cp .env.example .env
    ```
    Укажите необходимые переменные (пароли, ключи)
3. **Развертывание инфраструктуры:**
Убедитесь, что установлен и запущен Docker. Выполните команду:
    ```
    make all
    ```
    Команда автоматически соберет все образы и поднимет контейнеры.
4. **Доступ к сервисам:**
После сборки и инициализации сервисы будут доступны по следующим адресам:
    - **Airflow**: http://localhost:8080
    - **MetaBase**: http://localhost:3000
    - **MinIO Console**: http://localhost:9001
5. **Запуск ETL-процессов:**
В интерфейсе [Airflow](http://localhost:8080) активируйте все DAG. Проход пайплайна может занимать 30-40 минут, так как включает в себя полный цикл парсинга данных, их очистку, типизацию и последующую загрузку в DWH. После завершения ETL вы можете зарегистрироваться в [MetaBase](http://localhost:3000) и приступить к созданию дашбордов.
